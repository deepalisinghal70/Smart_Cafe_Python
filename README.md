# Smart Cafe: Interactive Core Python Logic Simulation

An interactive terminal-based cafe billing and inventory management simulation built using core Python data structures. This project transitions foundational syntax into a functional, logic-driven command-line application.

## 🚀 Features & Technical Implementations

* **Dynamic Inventory Tracking:** Utilizes Python `Dictionaries` to map items with real-time stock limits and pricing structures.
* **Control Flow:** Implements infinite `while` loops to maintain a seamless user shopping cart experience until an explicit checkout condition is met.
* **Robust Input Validation:** Leverages conditional logic (`if-elif-else`) to verify item availability, handle out-of-stock scenarios, and gracefully manage incorrect user inputs without crashing.
* **Order Analytics:** Tracks items dynamically using Python `Lists` to calculate precise billing details upon checkout.

## 🛠️ Tech Stack

* **Language:** Python 3
* **Environment:** Jupyter Notebook (`.ipynb`)

## 💻 Sample Logic Structure

* **Variables & Conditions:** Data tracking and verification using `in` keyword diagnostics.
* **Loops:** Conditional iteration execution using `break` statements for safe exit operations.
